# 🌟 Personal Portfolio Site

A modern, responsive portfolio website showcasing my projects, skills, and experience as a Computer Science student at Florida Atlantic University.

## ✨ Features

- **Responsive Design** - Optimized for all devices (mobile, tablet, desktop)
- **Dark/Light Mode** - Toggle between themes with smooth transitions
- **Smooth Animations** - Enhanced user experience with CSS transitions
- **Modern UI/UX** - Clean, professional design with intuitive navigation
- **Project Showcase** - Dedicated section highlighting key projects
- **Skills Display** - Visual representation of technical skills and tools
